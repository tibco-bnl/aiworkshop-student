# Student Workstation Requirements #

The requirements for the student workstation are:
* multiple students need to be able to work in an isolated desktop environment
* primairy development will performed inTIBCO Flogo
* development environment contains:
* * Microsoft Visual Code
* * VSCode Flogo extention
* * Golang
* * GCC
* * Google Chrome
* * Flogo extentions:
* * * as_metadata (https://github.com/matboek/as_metadata)
* * * langChain (https://github.com/mbloomfi-tibco/extensions/langChain)
* * * asVectorDB (https://github.com/mbloomfi-tibco/extensions/asVectorDB)
* remote desktop access
